# @evinova-demo/card

Content card with an optional title, styled with [`@evinova-demo/theme`](https://bit.cloud/evinova-demo) tokens.

## Install

```bash
pnpm add @evinova-demo/card
```

## Usage

```tsx
import { Card } from '@evinova-demo/card';

<Card title="Project overview">
  <p>Everything in this card inherits the shared design tokens.</p>
</Card>
```
